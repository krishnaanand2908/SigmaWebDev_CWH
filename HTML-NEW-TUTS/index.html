HTML Definition List

HTML
Hypertext Markup Language, HTML, is a markup language which is used to provide a framework to any website. It is like a skeleton for a website. The word Hypertext means linking of text with other documents and Markup Language means denotes a language utilising specific tags.
CSS
Cascading Style Sheets, CSS, is a stylesheet language which is used to provide an html document styles and beauty. It is like a beauty filter for a website.
JavaScript
JavaScript, JS, is a scripting language used to provide functionality to an html document. It is also known as the language of the internet or language of the web as it is a widely used language for creating websites. 50% of the world's programmers use JavaScript as their main programming language.
