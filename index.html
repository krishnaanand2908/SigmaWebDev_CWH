What is Web Developement?

  Hello readers!
  Web Developement refers to the developement of websites and web based applications!
  A basic website consists of HTML (Hypertext Markup Language), CSS (Cascade Style Sheets) and JS (Javascript)
  HTML provides a basic skeleton to the website, CSS beautifies the website by adding colors, backgrounds, etc whereas Javascript makes the   website functioning.
