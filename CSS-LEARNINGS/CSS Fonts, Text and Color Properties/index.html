In CSS colors can be represented in the following ways:

Name Brief Description Examples
Color Keywords Color Keywords are basically the name of the colors. Red, Aqua, Cyan, Crimson Red, Hot Pink, etc.
HEX In this format, we write a pound(#) followed by 6 letters or numbers. #FFFFFF, #FF0000, #00FFFF, etc.
RGB RGB values tells us about how much red, green and blue is in the color (ranging from 0 to 255) rgb(0, 0, 0); rgb(255, 255, 255); rgb(0, 100, 100); etc.
RGBA RGB along with an aplha value. The aplha value ranges from 0 to 1 and tells us the opacity of the color. rgba(0, 0, 0, 1); rgba(255, 255, 255, 0.5); rgba(0, 100, 100, 0); etc.
HSL HSL(Hue, Saturation, Lightness) are alternative representations of the RGB color model hsl(360, 100%, 100%); hsl(30, 70%, 100%); hsl(29, 100%, 60%); etc.
HSLA HSL with aplha where alpha ranges between 0 and 1 telling about the opacity of the color, hsla(360, 100%, 100%, 1); hsla(30, 70%, 100%, 0.7); hsla(29, 100%, 60%, 0); etc.
